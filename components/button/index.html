Button 按钮

常用的操作按钮。

基础用法

使用 type、plain、round 和 circle 来定义按钮的样式。

禁用状态

你可以使用 disabled 属性来定义按钮是否被禁用。

图标按钮

使用 icon 属性来定义按钮的图标。

按钮组

使用 <PxButtonGroup> 对多个按钮分组。
